Simple, Powerful API

Three endpoints. One API key. Download from anywhere.

GET /api/download Download media from any supported platform
PARAMTYPEDESCRIPTION
urlstringThe social media video/post URLREQUIRED
apikeystringYour GlenTechKenya API keyREQUIRED
GET https://glentechkenya-api.vercel.app/api/download?url=https://www.tiktok.com/@user/video/123456&apikey=GTKAPI-FREE-001 // You can also pass apikey as a header: x-api-key: GTKAPI-FREE-001
GET /api/keygen Generate a new free API key
PARAMTYPEDESCRIPTION
namestringYour name or app nameREQUIRED
emailstringYour email addressREQUIRED
GET https://glentechkenya-api.vercel.app/api/keygen?name=Glen&email=glen@example.com
GET /api/index API health check & info
